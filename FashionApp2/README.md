# React Fashion Store (FashionApp2)

This folder contains the runnable React storefront built with Vite, React Router, Redux Toolkit, and CSS Modules.

## Run The App

From the repository root:

```bash
cd FashionApp2
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build

```bash
cd FashionApp2
npm run build
```

## What Is Included

- Multi-page storefront: Home, Shop, About Us, Customer Care
- Shopping bag flow with Redux Toolkit
- Cart persistence with localStorage
- Reusable components with CSS Modules

## Tech Stack

- React 18
- Vite
- React Router DOM
- Redux Toolkit + React Redux
- react-multi-carousel
